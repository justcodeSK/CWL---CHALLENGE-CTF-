# Introduction

This project demonstrates practical cybersecurity skills through a set of Capture The Flag (CTF) style challenges involving multiple domains of security analysis.

The objective of this assessment was to showcase the ability to perform:

- Open Source Intelligence (OSINT)
- Reverse Engineering
- Digital Forensics
- Security Analysis

Each challenge required identifying hidden information, analyzing files, or reverse engineering software components to uncover sensitive data.

The methodology followed includes:

- Systematic investigation
- Tool-based analysis
- Documentation of every step
- Evidence-based conclusions
